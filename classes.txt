Class: Boat

Attributes:
name (string)
fuelRemaining (integer)
engineOn (boolean)
lightsOn (array)
passengers (array)

Methods:
rename (sets name string to user input)
refuel (sets fuelRemaining to 100)
doIgnitionSequence (changes engineOn to true)
turnLightsOn (takes user input array of index positions and sets those lightsOn array index position boolean values to true)
addPassenger (adds user input string to end of passengers array)
